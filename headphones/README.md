# Headphones Landing Page

## Overview

This project is a responsive **Headphones Landing Page** designed using **HTML** and **CSS**.  
The design comes from a Figma file and focuses on clean aesthetics, usability, and responsive layouts for both desktop and mobile devices.

## Features

- **Responsive Design**: Adapts to different screen sizes. Mobile layout activates at **480px or less**.
- **Interactive Elements**:
  - Links hover/active color: `#FF6565`
  - Buttons hover/active opacity: `0.9`
- **Content Layout**:
  - Maximum width: **1000px** centered
  - Clean typography with **Source Sans Pro** and **Spin Cycle OT**

## Technologies Used

- HTML5
- CSS3
- Google Fonts: [Source Sans Pro](https://fonts.google.com/specimen/Source+Sans+Pro)
- Custom Font: Spin Cycle OT
